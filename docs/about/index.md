# 关于本站

## 📖 博客简介

这是一个专注于分享 **Coding Agent 在学术科研中的应用经验**的技术博客。

在从事学术研究的过程中，我逐渐认识到 Coding Agent 工具的强大潜力。无论是文献调研、数据分析、代码实现还是论文写作，Coding Agent 都能显著提升效率。然而目前缺少系统的、结合学术场景的深度实践指南，因此我创建了这个博客，**系统地记录和分享实践经验**。

## 🎯 内容定位

本博客涵盖以下核心主题：

- **🤖 Agent 基础** — 深入理解 LLM Agent 的核心概念、架构原理和工作机制
- **💻 Coding Agent 实践** — Claude Code、OpenCode 等工具的深度使用技巧
- **🔬 学术科研案例** — Coding Agent 在文献调研、数据分析、论文写作等场景中的真实应用
- **🛠️ 工具对比评测** — 主流 Coding Agent 工具的横向对比与选型建议
- **💡 经验心得分享** — 实践总结、踩坑记录、效率技巧

## 🛠️ 技术栈

- **VitePress** — 静态站点生成器
- **GitHub Pages** — 站点托管与自动化部署
- **KaTeX / Mermaid** — 数学公式与图表渲染
